# 介绍

移动端项目跨平台框架采用[Uniapp](https://uniapp.dcloud.net.cn/)，UI框架采用[UView-Plus](https://uiadmin.net/uview-plus/)，项目使用`cli`脚手架的方式进行搭建及构建。

基于`monorepo`架构的优势，移动端得以复用PC端项目中的网络请求、状态存储、通用接口、工具等公共模块，同时为了提供更加符合移动端操作习惯的UI组件，并对一些无法兼容小程序平台的公共模块进行适配，所以创建了专为移动端打造的`liv-uni`库，提供了大量开箱即用的移动端组件及工具方法用于快速实现相关业务。

## 公共组件

根据移动端屏幕比例及操作习惯，封装了字典选择器、网格选择器、通用列表、通用表单等组件。

## 工具库

根据移动端相关特性，封装了移动端专用的小程序登录、图片预览、文件上传等方法。

## Hooks

封装了大量移动端定位、录音等常用功能，内部对H5及小程序等平台进行了兼容处理，支持跨平台使用。
